# C/C++ Coding Exercise
A web server that can read in JSON and return it as XML.

### Requirements
1) Boost - sudo apt-get install libboost-all-dev
2) gcc 4.2.x+
3) Linux/Unix or MacOS X

### Use
1) ```$ make```
2) ```$ ./run_server.sh```
3) ```$ ./client```

### References
1) nweb - https://github.com/ankushagarwal/nweb
2) boost - https://www.boost.org/
